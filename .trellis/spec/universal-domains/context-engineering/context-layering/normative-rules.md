# Normative Rules

* Stable cross-project rules must be separated from project-specific and task-specific context.
* Task context must not overwrite higher-priority stable context unless the workflow explicitly allows it.
* Temporary working notes and transient observations must not be promoted to stable context without review.
* Derived summaries must identify what source layer they were derived from.
* Context composition must preserve traceability between source layer and injected content.
