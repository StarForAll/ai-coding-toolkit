# Normative Rules

* Manual verification should define what to do, what to observe, and what would count as failure before execution begins.
* A manual check must not be described as strong evidence if it is vague, non-repeatable, or missing expected outcomes.
* Manual verification scope should reflect the actual change risk rather than serving as a token sign-off step.
* Human-performed checks should preserve notable findings, gaps, and uncertainties rather than collapsing everything into a simple pass claim.
* When manual verification is the main remaining evidence source, readiness must be reviewed with extra care.
