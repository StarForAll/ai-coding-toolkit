# Normative Rules

* Production issue reproduction must begin from observed evidence rather than reconstructed guesses alone.
* Reproduction attempts should preserve live-system safety and avoid adding avoidable user impact.
* The reproduction environment should match the relevant production conditions closely enough to make conclusions meaningful.
* Failure to reproduce locally must not be treated as proof that the production issue is invalid.
* Uncertainty about environment fidelity, triggering conditions, or evidence gaps must remain explicit during investigation.
