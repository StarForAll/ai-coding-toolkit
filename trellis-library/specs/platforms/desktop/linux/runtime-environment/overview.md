# Linux Runtime Environment

## Purpose

Define reusable Linux-specific desktop rules for environment-driven runtime
behavior such as session variables, display-server assumptions, XDG directory
conventions, and other user-environment differences that affect correctness.

## Applicability

Use this concern when desktop software runs on Linux and behavior depends on
environment variables, user-session context, writable directories, or display
stack assumptions that can vary across distributions or desktop sessions.
