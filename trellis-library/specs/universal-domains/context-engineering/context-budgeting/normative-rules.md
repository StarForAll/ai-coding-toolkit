# Normative Rules

* Context should be budgeted intentionally rather than allowed to grow without bound.
* Higher-value and higher-stability context must take priority over low-value history.
* When context pressure rises, workflows should summarize or compress lower-priority material rather than silently dropping critical constraints.
* Large raw histories should not be injected when a validated summary can preserve decision-relevant facts.
* Context-budget tradeoffs must be explicit when important information is omitted.
