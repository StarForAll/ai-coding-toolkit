# Normative Rules

* A release decision must be tied to explicit readiness evidence, not schedule pressure alone.
* Release controls should scale with blast radius, uncertainty, and reversibility.
* Incomplete verification or unresolved operational risks must be visible at release decision time.
* Release scope should remain understandable and bounded rather than silently accumulating unrelated changes.
* Changes that cannot be safely observed, halted, or reversed must be treated as higher-risk releases.
