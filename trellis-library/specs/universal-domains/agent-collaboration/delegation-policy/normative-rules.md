# Normative Rules

* Work should be delegated only when the task boundary is concrete enough to avoid ambiguous ownership.
* Delegation must not hide unresolved requirement ambiguity or missing evidence.
* The delegating agent must provide enough context for the delegate to act without inventing hidden assumptions.
* Tasks that are tightly coupled to immediate blocking decisions should remain local unless a clear handoff boundary exists.
* Delegation must be revisited when the coordination cost exceeds the execution benefit.
