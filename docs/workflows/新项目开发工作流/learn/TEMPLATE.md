## 起草阶段（AI 或人工填写）

### 现象

- **发生了什么**：<具体行为描述，不要先下结论>

可直接按下面这种口吻写：

- "我本来以为……，结果……"

### 详细描述（可选）

按真实开发顺序补充：当时发生了什么 → 为什么卡住 → 怎么发现并修正的。

能一句话说清的地方不要硬写成长篇。

### 建议改哪里

- "把这句话挪到前面"
